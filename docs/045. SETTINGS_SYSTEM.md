  - Traditional German format with photo
  - Best for: Germany, Austria, Switzerland
- **Compact 1-Page**
  - Concise single-page format
  - Best for: Entry level, Career changers

**Impact:**
- Resume editor loads selected template by default
- Provides format-specific tips
- Adjusts export options

  ADD COLUMN timezone text,

  ADD COLUMN target_role text,
  ADD COLUMN target_industry text,
  ADD COLUMN experience_level text,
  ADD COLUMN job_search_intensity text DEFAULT 'moderate',
  ADD COLUMN resume_format_preference text DEFAULT 'ats_standard',

  ADD COLUMN email_notifications boolean DEFAULT true,
