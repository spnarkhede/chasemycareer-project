## Step 1: Supabase Setup (10 minutes)

### 1.1 Create Project (2 minutes)

1. Go to https://supabase.com/dashboard
2. Click "New Project"
3. Fill in:
   - Name: `chase-my-career`
   - Database Password: (generate strong password - save it!)
   - Region: (closest to your users)
4. Click "Create new project"
5. Wait ~2 minutes

### 1.2 Get Credentials (1 minute)

1. Go to Project Settings ‚Üí API
2. Copy and save:
   ```
   Project URL: https://xxxxx.supabase.co
   anon key: eyJhbGc...
   ```

### 1.3 Run Database Setup (5 minutes)

1. Go to SQL Editor
2. Click "New query"
3. Copy SQL from: `docs/SUPABASE_SETUP.md`
4. Run these sections in order:
   - **Section 2.2:** Initial schema (all tables)
   - **Section 2.3:** 50-day curriculum data
   - **Section 3.1:** Database functions
   - **Section 3.2:** Triggers
   - **Section 4.2:** RLS policies

5. Verify:
```sql
SELECT COUNT(*) FROM daily_tasks_template;
-- Should return 50
```

### 1.4 Configure Authentication (2 minutes)

1. Go to Authentication ‚Üí URL Configuration
2. Add Redirect URLs:
   ```
   http://localhost:5173/**
   https://www.chasemycareer.com/**
   https://chasemycareer.com/**
   ```

**‚úÖ Supabase Setup Complete!**

## Step 2: Deploy to Vercel (5 minutes)

### 2.1 Push to GitHub (2 minutes)

```bash
cd /workspace/chasemycareer

# Initialize git (if not done)
git init
git add .
git commit -m "Ready for production"
git branch -M main

# Create GitHub repo and push
# (Create repo at https://github.com/new first)
git remote add origin https://github.com/YOUR_USERNAME/chase-my-career.git
git push -u origin main
```

### 2.2 Import to Vercel (3 minutes)

1. Go to https://vercel.com/dashboard
2. Click "Add New" ‚Üí "Project"
3. Import your GitHub repository
4. Configure:
   - Framework: **Vite**
   - Root Directory: **frontend**
   - Build Command: **npm run build**
   - Output Directory: **dist**

5. Add Environment Variables:
   ```
   VITE_SUPABASE_URL=https://xxxxx.supabase.co
   VITE_SUPABASE_ANON_KEY=eyJhbGc...
   ```
   (Add for Production, Preview, Development)

6. Click "Deploy"
7. Wait ~2 minutes

**‚úÖ Your app is live at: https://your-project.vercel.app**

## Step 3: Configure Domain (10 minutes)

### 3.1 Purchase Domain (if needed) (5 minutes)

1. Go to Namecheap, Google Domains, or Cloudflare
2. Search: `chasemycareer.com`
3. Purchase (~$15/year)
4. Add privacy protection

### 3.2 Add Domain to Vercel (2 minutes)

1. In Vercel Dashboard ‚Üí Your Project ‚Üí Settings ‚Üí Domains
2. Click "Add"
3. Enter: `chasemycareer.com`
4. Click "Add"
5. Enter: `www.chasemycareer.com`
6. Click "Add"

### 3.3 Configure DNS (3 minutes)

Go to your domain registrar's DNS settings and add:

**A Record:**
```
Type: A
Name: @
Value: 76.76.21.21
TTL: 3600
```

**CNAME Record:**
```
Type: CNAME
Name: www
Value: cname.vercel-dns.com
TTL: 3600
```

**Save changes**

### 3.4 Wait for Propagation

- Initial verification: 5-10 minutes
- Full propagation: 24-48 hours
- Check status: https://www.whatsmydns.net/

**‚úÖ Domain configured!**
