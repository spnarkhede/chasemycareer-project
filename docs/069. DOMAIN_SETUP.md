## Part 2: DNS Configuration

### Scenario A: Using Vercel

#### Step 1: Add Domain in Vercel

1. Go to Vercel Dashboard
2. Select your project (chase-my-career)
3. Go to Settings ‚Üí Domains
4. Click "Add"
5. Enter: `chasemycareer.com`
6. Click "Add"
7. Repeat for: `www.chasemycareer.com`

#### Step 2: Configure DNS Records

Vercel will show you the required DNS records. Go to your domain registrar's DNS management and add:

**For Root Domain (chasemycareer.com):**
```
Type: A
Name: @ (or leave empty)
Value: 76.76.21.21
TTL: 3600 (or Auto)
```

**For WWW Subdomain (www.chasemycareer.com):**
```
Type: CNAME
Name: www
Value: cname.vercel-dns.com
TTL: 3600 (or Auto)
```

#### Step 3: Verify Configuration

1. Wait 5-10 minutes for DNS propagation
2. Vercel will automatically verify the domain
3. SSL certificate will be provisioned automatically
4. You'll see a green checkmark when ready

## Part 4: Domain Redirects

### Redirect Root to WWW

**Why?**
- Consistency in URLs
- Better for SEO
- Cleaner analytics

#### Vercel Configuration

In `vercel.json`:
```json
{
  "redirects": [
    {
      "source": "https://chasemycareer.com/:path*",
      "destination": "https://www.chasemycareer.com/:path*",
      "permanent": true
    }
  ]
}
```

#### Cloudflare Configuration

1. Go to Rules ‚Üí Page Rules
2. Click "Create Page Rule"
3. URL: `chasemycareer.com/*`
4. Setting: Forwarding URL
5. Status Code: 301 (Permanent Redirect)
6. Destination: `https://www.chasemycareer.com/$1`
7. Save

Or use Bulk Redirects (better):
1. Go to Rules ‚Üí Redirect Rules
2. Create rule:
   - If: Hostname equals `chasemycareer.com`
   - Then: Dynamic redirect
   - Expression: `concat("https://www.chasemycareer.com", http.request.uri.path)`
   - Status code: 301

## Part 12: Troubleshooting

### Domain Not Resolving

**Check:**
1. DNS records are correct
2. Nameservers are updated (if using Cloudflare)
3. Wait 24-48 hours for full propagation
4. Clear DNS cache: `ipconfig /flushdns` (Windows) or `sudo dscacheutil -flushcache` (Mac)

### SSL Certificate Error

**Solutions:**
1. Wait for certificate provisioning (can take 24 hours)
2. Verify DNS records are correct
3. Check CAA records don't block Let's Encrypt
4. Contact hosting support

### Email Not Working

**Check:**
1. MX records are correct
2. SPF record is set
3. DKIM is configured
4. Wait 24 hours for DNS propagation
5. Test with mail-tester.com

### Redirect Loop

**Solutions:**
1. Check redirect rules aren't conflicting
2. Verify SSL mode is correct (Full Strict for Cloudflare)
3. Clear browser cache
4. Check `.htaccess` or `_redirects` file
