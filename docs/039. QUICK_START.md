## Common Issues & Solutions

### Issue 1: "Cannot find module" errors
**Solution:**
```bash
cd frontend
rm -rf node_modules package-lock.json
npm install
```

### Issue 2: Database connection errors
**Solution:**
- Verify `.env` file has correct Supabase credentials
- Check Supabase project is active
- Verify API keys are correct

### Issue 3: No tasks created after registration
**Solution:**
- Check if `daily_tasks_template` has 50 rows
- Verify trigger is enabled:
```sql
SELECT * FROM pg_trigger 
WHERE tgname = 'trigger_auto_initialize_tasks';
```

### Issue 4: Admin dashboard shows "Access denied"
**Solution:**
- Verify you added yourself to `admin_users` table
- Check the user_id matches your auth user ID
- Log out and log back in

### Issue 5: Google OAuth not working
**Solution:**
- Google OAuth requires valid client ID
- For local testing, you can skip OAuth and use email/password
- To enable: Get Google OAuth credentials from Google Cloud Console
