## ğŸ› Common Issues

### Issue: "Function Not Found"

**Cause:** Edge Function not deployed

**Fix:**
```bash
# Redeploy the Edge Function
supabase functions deploy create-test-users
```

### Issue: "Email Already Exists"

**Cause:** Test users already exist

**Fix:**
- Just click "Create Test Users" again
- Function automatically deletes old users first
- New users created with same credentials

### Issue: Can't Log In After Creation

**Possible Causes:**
1. Wrong password (check Caps Lock)
2. Wrong email (check for typos)
3. Users not actually created (check for errors)

**Fix:**
1. Double-check credentials
2. Recreate users
3. Check browser console for errors
4. Check Supabase dashboard for users

### Issue: Admin User Has User Role

**Cause:** Profile update failed

**Fix:**
1. Check Edge Function logs
2. Verify profiles table exists
3. Manually update role in Supabase dashboard:
   ```sql
   UPDATE profiles 
   SET role = 'admin' 
   WHERE email = 'admin@jobtracker.test';
   ```
