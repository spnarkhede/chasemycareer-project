## ğŸ” Troubleshooting

### Problem: "Create Test Users" button doesn't work

**Solution:**
1. Open browser console (F12)
2. Check for error messages
3. Verify you have internet connection
4. Try refreshing the page and clicking again

### Problem: Login fails with "Invalid credentials"

**Possible causes:**
1. **Test users not created yet**
   - Go back to Step 1 and create the users first

2. **Wrong password**
   - Make sure you're copying the password exactly
   - Password is case-sensitive
   - Check for extra spaces

3. **Wrong email**
   - Use `admin@jobtracker.test` or `user@jobtracker.test`
   - No spaces before or after

### Problem: "Too many failed attempts"

**Solution:**
- Wait 15 minutes before trying again
- This is a security feature to prevent brute force attacks

### Problem: Redirected to wrong page after login

**Solution:**
1. Clear browser cache
2. Try logging out and logging in again
3. Check browser console for errors
