## ðŸ› Common Issues

### Email Not Received?

**Check:**
1. Spam/junk folder
2. Email address is correct
3. Wait 2-3 minutes
4. Try requesting again

---

### Invalid Token Error?

**Reasons:**
- Link expired (>1 hour old)
- Link already used
- Invalid URL

**Solution:**
- Request new reset link

---

### Password Validation Error?

**Check your password has:**
- 8+ characters
- Uppercase letter
- Lowercase letter
- Number
- Special character

## ðŸ’¡ Pro Tips

### For Users

1. **Use a Password Manager**
   - Generates strong passwords
   - Remembers them for you
   - More secure

2. **Check Spam Folder**
   - Reset emails sometimes go to spam
   - Mark as "Not Spam" for future

3. **Act Quickly**
   - Links expire in 1 hour
   - Don't wait too long

4. **Keep Email Secure**
   - Anyone with access to your email can reset your password
   - Use 2FA on your email account
