## ğŸš€ How to Run the Project

### Prerequisites
- Node.js 18+ installed
- Supabase account
- Environment variables configured

### Setup Steps

1. **Clone and Install**
```bash
cd frontend
npm install
```

2. **Configure Environment Variables**
Create `.env` file in frontend directory:
```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
VITE_GOOGLE_CLIENT_ID=your_google_client_id
```

3. **Run Development Server**
```bash
npm run dev
```

4. **Access Application**
- Frontend: http://localhost:5173
- Login with test account or create new account

### Creating an Admin User

To make a user an admin, run this SQL in Supabase:

```sql
-- Replace 'user-id-here' with actual user ID
INSERT INTO admin_users (user_id, role, permissions)
VALUES (
  'user-id-here',
  'admin',
  '{"view_users": true, "edit_users": true, "delete_users": false, "send_messages": true}'::jsonb
);
```

When a user registers, a database trigger automatically:

- Copies content from daily_tasks_template
- Links tasks to the new user
