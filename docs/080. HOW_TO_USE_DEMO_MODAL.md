# How to Use the Demo Modal

## For End Users

### Accessing the Demo
1. Visit the Chase My Career landing page
2. Click the **"View Demo"** button in the hero section
3. A modal window will open with a complete tool overview

### What You'll See

#### Tool Overview
Learn about Chase My Career's comprehensive 50-day job search program with structured guidance and proven strategies.

#### Key Features (6 Main Features)
- **50-Day Structured Program**: Follow a proven day-by-day plan
- **Sequential Learning**: Build skills progressively
- **Actionable Tasks**: Get clear, step-by-step instructions
- **Progress Tracking**: Monitor your advancement
- **Job Application Tools**: Track applications and interviews
- **Interview Practice**: Prepare with STAR stories

#### Program Structure (8 Weeks)
- **Week 1-2**: Foundation & Resume Building
- **Week 3-4**: LinkedIn & Online Presence
- **Week 5-6**: Networking & Outreach
- **Week 7-8**: Applications & Follow-ups

#### What You'll Get (8 Benefits)
✓ Daily actionable tasks with clear instructions
✓ Resume and LinkedIn optimization guides
✓ Networking templates and outreach strategies
✓ Interview preparation and practice tools
✓ Job application tracking system
✓ Progress tracking and milestone celebrations
✓ Company research and targeting tools
✓ Salary negotiation and follow-up guides

#### How It Works (4 Steps)
1. **Start Your Journey**: Sign up and complete onboarding
2. **Follow Daily Tasks**: Complete structured activities
3. **Track Your Progress**: Monitor all activities in one place
4. **Land Your Dream Job**: Complete the program and celebrate

### Taking Action

After reviewing the overview, you have two options:

#### Option 1: Try Interactive Demo
- Click the **"Try Interactive Demo"** button
- You'll be logged into a demo account with sample data
- Explore all features without creating an account
- See how the platform works in real-time

#### Option 2: Sign Up Free
- Click the **"Sign Up Free"** button
- Create your own account to start your journey
- Begin your personalized 50-day program
- Track your real job search progress

### Closing the Modal
- Click the **X** button in the top right
- Click outside the modal window
- Press the **Escape** key on your keyboard

## For Developers

### Component Location
```
src/components/common/DemoModal.tsx
```

### Usage Example
```typescript
import { DemoModal } from '@/components/common/DemoModal';

function YourComponent() {
  const [isOpen, setIsOpen] = useState(false);

  return (
    <>
      <Button onClick={() => setIsOpen(true)}>
        View Demo
      </Button>
      
      <DemoModal 
        open={isOpen} 
        onOpenChange={setIsOpen} 
      />
    </>
  );
}
```

### Props Interface
```typescript
interface DemoModalProps {
  open: boolean;              // Controls modal visibility
  onOpenChange: (open: boolean) => void;  // Callback when modal state changes
}
```

## Troubleshooting

### Modal Won't Open
- Check that state is properly managed
- Verify button onClick handler is connected
- Ensure DemoModal component is rendered

### Demo Login Fails
- Verify demo account credentials are correct
- Check Supabase connection
- Review error messages in console
- Ensure proper error handling is in place

### Content Not Displaying
- Check for TypeScript errors
- Verify all imports are correct
- Ensure components are properly installed
- Review browser console for errors

### Styling Issues
- Verify Tailwind CSS is configured
- Check that shadcn/ui components are installed
- Ensure color tokens are defined in theme
- Test on different screen sizes
