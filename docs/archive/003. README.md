## ðŸš€ Quick Start

### Prerequisites
- Node.js 18 or higher
- npm or pnpm package manager
- Supabase account (free tier works)

### Installation

```bash
# Clone the repository
cd /workspace/chasemycareer

# Navigate to frontend
cd frontend

# Install dependencies
npm install

# Create .env file
cat > .env << 'EOF'
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
VITE_GOOGLE_CLIENT_ID=your_google_client_id
EOF

# Start development server
npm run dev
```

Visit http://localhost:5173 to see the application.

## ðŸ—„ï¸ Database Schema

### Core Tables
- **profiles** - User profile information
- **daily_tasks** - User's 50-day task instances
- **daily_tasks_template** - Master curriculum (50 days)
- **job_applications** - Application tracking
- **interviews** - Interview management
- **networking_contacts** - Professional network
- **documents** - Resume/cover letter storage

### Admin Tables
- **admin_users** - Admin role assignments
- **user_messages** - Admin messaging
- **user_activity** - Activity logging
- **achievements** - Milestone tracking

### Database Functions
- `initialize_user_daily_tasks()` - Auto-create 50 tasks for new users
- `get_user_statistics()` - Get user progress metrics
- `get_admin_dashboard_stats()` - Platform-wide analytics
- `is_admin()` - Check admin permissions

Days 36-42: Salary negotiation, questions to ask, virtual setup, body language, follow-up
