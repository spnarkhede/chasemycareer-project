## ğŸ“ Content Fields Explained

### `dayNumber` (number)
- The day number (1-50)
- **Do not change this** - it must match the file name

### `theme` (string)
- A short, catchy title for the day
- Example: "Resume: Build Your Wins Bank"
- Keep it under 50 characters for best display

### `objective` (string)
- A single sentence describing the main goal for the day
- Example: "Create a comprehensive list of measurable achievements from your past roles."
- Keep it clear and actionable

### `keyTasks` (array of strings)
- 3-5 bullet points of the main tasks for the day
- Each should be specific and actionable
- Example:
  ```typescript
  keyTasks: [
    'List 10 measurable achievements from past roles',
    'Include metrics: % improvements, time saved, volume managed',
    'Identify 3 skill gaps you want to close',
  ]
  ```

### `instructions` (array of objects)
- Detailed step-by-step instructions
- Each object has:
  - `title`: Section heading
  - `steps`: Array of detailed steps
- You can have multiple instruction sections
- Example:
  ```typescript
  instructions: [
    {
      title: 'Document Your Achievements',
      steps: [
        'Review your work history and identify 10 significant accomplishments',
        'For each achievement, quantify the impact with specific metrics',
        'Include percentage improvements, time saved, volume managed',
      ],
    },
    {
      title: 'Identify Skill Gaps',
      steps: [
        'Review job descriptions for your target roles',
        'List skills that appear frequently but you lack',
        'Prioritize 3 skill gaps to address',
      ],
    },
  ]
  ```

### `resources` (array of objects)
- Links to helpful tools, templates, or articles
- Each object has:
  - `title`: Name of the resource
  - `url`: Link to the resource
- Can be an empty array `[]` if no resources
- Example:
  ```typescript
  resources: [
    {
      title: 'Resume Template Library',
      url: 'https://example.com/templates',
    },
    {
      title: 'LinkedIn Salary Calculator',
      url: 'https://www.linkedin.com/salary/',
    },
  ]
  ```

### `timeCommitment` (string)
- Estimated time to complete the day's tasks
- Examples: "1-2 hours", "2-3 hours", "30 minutes"
- Be realistic about the time required

### `proTip` (string)
- A single expert tip or insight
- Should be practical and actionable
- Example: "Focus on results, not responsibilities. Employers want to see what you achieved, not just what you did."

## ğŸ¯ Content Guidelines

### Writing Style
- **Clear and Direct**: Use simple, straightforward language
- **Actionable**: Every instruction should be something the user can do
- **Specific**: Include concrete examples and metrics
- **Encouraging**: Maintain a supportive, motivational tone

### Formatting Tips
- Use proper punctuation and grammar
- Keep sentences concise
- Use bullet points for lists
- Include examples where helpful
- Break complex tasks into smaller steps

### Content Length
- **Theme**: 30-50 characters
- **Objective**: 1-2 sentences
- **Key Tasks**: 3-5 items
- **Instructions**: 2-4 sections with 3-6 steps each
- **Resources**: 0-5 links
- **Pro Tip**: 1-2 sentences

## ğŸ“ Need Help?

### Common Issues

**Q: I edited a day but don't see changes**
A: Make sure you saved the file and the application has reloaded

**Q: I get a TypeScript error**
A: Check that your content structure matches the `DayContent` type exactly

**Q: How do I add more instruction sections?**
A: Just add more objects to the `instructions` array:
```typescript
instructions: [
  { title: 'Section 1', steps: ['Step 1', 'Step 2'] },
  { title: 'Section 2', steps: ['Step 1', 'Step 2'] },
  { title: 'Section 3', steps: ['Step 1', 'Step 2'] },
]
```

**Q: Can I add images or videos?**
A: Currently, the structure supports text and links only. You can link to external images/videos in the resources section.
